# 智能记账本 APP

这是一个用Python实现的智能记账本应用，具有收支记录管理、收支预测和经济水平画像等功能。

## 功能特点

### 1. 经济收支记录
- **收支信息录入**：用户可以手动逐条录入收入和支出信息，包含来源/去向、金额、时间等
- **记录编辑与删除**：支持删除已有记录，通过ID唯一标识
- **收支记录展示**：以表格形式展示所有收支记录

### 2. 收支预测
- **未来收支预测**：基于历史数据使用线性回归模型预测未来30天的收入和支出情况
- **日期范围筛选**：可以指定时间范围进行预测分析

### 3. 经济水平画像
- **经济指标计算**：计算恩格尔系数、平均消费倾向(APC)、边际消费倾向(MPC)
- **经济分析报告**：根据计算结果生成用户经济水平分析报告

## 安装说明

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

## 使用指南

### 主界面
程序启动后会显示主界面，包含三个主要功能模块按钮：
- **收支编辑**：进入收支记录管理界面
- **收支预测**：进入收支预测分析界面
- **经济画像**：查看个人经济水平分析

### 收支编辑
1. 点击"新增记录"按钮添加收支信息
2. 点击"删除记录"按钮删除指定ID的记录
3. 表格中显示所有已添加的收支记录

### 收支预测
1. 可选：输入开始日期和结束日期（格式：YYYY-MM-DD）
2. 点击"开始预测"按钮生成未来30天的收支预测
3. 图表展示预测结果，下方显示汇总信息

### 经济画像
1. 自动分析所有收支数据
2. 显示恩格尔系数、APC、MPC等经济指标
3. 提供详细的经济分析说明

## 技术栈

- **Python 3.8+**
- **Pyside**：GUI界面开发
- **NumPy/Pandas**：数据处理
- **Scikit-learn**：机器学习预测
- **Matplotlib**：数据可视化

## 注意事项

1. 首次使用时，数据文件会自动创建
2. 预测功能需要一定量的历史数据才能提供准确结果
3. 恩格尔系数的计算基于描述中包含食品相关关键词的支出
4. 建议定期备份data目录中的数据文件
5. 如果GUI界面无法正常使用，会自动提供终端版的功能
